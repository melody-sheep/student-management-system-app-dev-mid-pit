1. PROJECT OVERVIEW
========================================================================
A full-stack web application for managing student records with the following
core functionalities:

□ Add student record
□ View all students
□ Edit student record
□ Delete student record
□ Search student by name
□ Filter by course

========================================================================
2. TECH STACK SUMMARY
========================================================================
Backend:
- Runtime: Node.js
- Framework: Express.js
- Database: MySQL
- Packages: express, mysql2, dotenv

Frontend:
- HTML5 (semantic elements)
- CSS3 (Grid, Flexbox, variables, transitions)
- JavaScript (Vanilla/Plain JS with Fetch API)

6. API ENDPOINTS CHECKLIST
========================================================================
All endpoints must:
- Use parameterized queries (NO string concatenation)
- Return JSON responses
- Use proper HTTP status codes (200, 201, 400, 404, 500)
- Handle errors gracefully

□ [ ] GET    /api/students              - Get all students (ordered by last_name ASC)
□ [ ] GET    /api/students/:id          - Get single student by ID
□ [ ] POST   /api/students              - Add new student (validate all fields)
□ [ ] PUT    /api/students/:id          - Update existing student
□ [ ] DELETE /api/students/:id          - Delete student
□ [ ] GET    /api/students/search/:keyword - Search students by name (LIKE %keyword%)
